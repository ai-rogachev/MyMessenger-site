Some Features


Simple Design

Simplest app design. There is nothing superfluous - everything you need to exchange messages and nothing more.


No history

The app does not save any history. Only in the current session, the chat history is available and only for application users. Only the current moment - only online communication.


Online only

The application provides messaging only for users with online status. No delayed messages - only the current moment... Ready to wait - write letters - this application is not for you ;)


Encryption

All messages between server and client encrypted by AES-256 with 256 bits key. Encryption key a creating in each login, new key in each login. This application did not guarantee that all messages will be secured. But security will be increased...
